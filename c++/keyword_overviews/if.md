### Syntax
```c++
if (boolean-expression) compound-statment else if (boolean-expression) compound-statment else compound-statement
```
### Control Flow
An if block specifies a block of code to execute when a condition is true.
### Example
```c++
int i = 0;
if (false)
	i--;
else if (false)
	i-=2;
else
	i++; // i is now 1
```
